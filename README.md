# Portfolio — Prakash Kumar Sarangi

Personal portfolio website built with vanilla HTML, CSS, and JavaScript.  
**Theme:** Aurora / Cosmic — dark navy base with glowing teal, violet, and gold accents.

## Features (Phase 1)
- Constellation preloader animation
- Hero section with typing animation
- Glassmorphism navigation
- Responsive design with mobile menu
- Aurora starfield background

## Tech Stack
- HTML5 / CSS3 / JavaScript (Vanilla)
- Google Fonts (Inter, Syne, JetBrains Mono)

## Setup
```bash
# Local development
npx serve .
```
